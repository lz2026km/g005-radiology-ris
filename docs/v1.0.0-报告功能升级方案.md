### 3.3 功能实现细节

#### 快捷键体系
```
F1: 显示帮助浮层（快捷键说明）
F2: 保存报告（Ctrl+S同步）
F3: 提交报告（需完整度检测通过）
F4: 打印预览弹窗
F5: 切换语音录入开关
F6: 打开模板选择器抽屉
F7: 打开词库选择器抽屉
F8: 打开历史报告面板
F9: 打开报告对比弹窗
F10: 刷新AI建议
F11: 全屏/退出全屏
F12: 显示报告质控面板
Ctrl+B: 加粗选中文字
Ctrl+I: 斜体（诊断意见）
Ctrl+Enter: 快速提交
```

#### 语音录入
- 按钮状态：麦克风图标（待机）/ 红色录音中 / 加载中
- 模拟转写：输入3秒后自动在光标处插入模拟文字
- 支持在所见、诊断、印象三处分别录入

#### AI自动填充
- 根据选择的检查设备（CT/MRI/DR）和部位
- 从PHRASE_LIBRARY自动提取正常描述作为模板
- 疑似阳性关键词高亮（红色加粗）

#### 报告完整度检测
- 所见字段：≥20字符
- 诊断意见：≥5字符
- 检查医师签名：必填
- 不满足时F3提交按钮变灰+tooltip提示

#### 时限提醒
- 急诊检查：30分钟黄色预警，60分钟红色报警
- 平诊检查：2小时黄色预警，4小时红色报警
- 住院检查：24小时黄色预警，48小时红色报警
