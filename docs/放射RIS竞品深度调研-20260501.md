## 五、技术实现方案

### AI报告辅助（纯前端模拟）
```typescript
// AI推荐逻辑
const aiRecommend = (examType: string, bodyPart: string, findings: string) => {
  // 基于模板+关键词生成推荐
  const templates = getTypicalFindings(examType, bodyPart)
  const conclusions = getTypicalConclusions(examType)
  return { findings: templates, conclusions }
}
```

### 语音录入（Web Speech API）
```typescript
const startVoiceInput = () => {
  const recognition = new webkitSpeechRecognition()
  recognition.continuous = true
  recognition.onresult = (event) => {
    const text = event.results[0][0].transcript
    insertText(text)
  }
  recognition.start()
}
```
