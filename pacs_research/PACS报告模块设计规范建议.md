## 一、整体配色方案

### 1.1 推荐配色体系：深色主题（医学影像工作站行业标准）

| 区域 | 推荐色值 | 用途说明 |
|------|---------|---------|
| 影像显示区背景 | `#0a0a0a` / `#111111` | 纯黑或近黑，最大化影像对比度 |
| 工具栏/侧边栏 | `#1e293b` (深蓝灰) | 沉稳专业，减少视觉疲劳 |
| 主色调 | `#1e3a5f` (深海蓝) | 医疗专业感、可信赖 |
| 主色调亮色 | `#2d5a8a` | 悬停/激活状态 |
| 报告编辑区背景 | `#ffffff` | 纯白背景确保文字可读性 |
| 卡片/面板背景 | `#f8fafc` | 浅灰，不刺眼 |
| 文字主色 | `#1e293b` | 深灰蓝，易读 |
| 文字次级 | `#64748b` | 中灰，用于标签/说明 |
| 成功/正常 | `#059669` | 绿色系 |
| 警告/待审 | `#d97706` | 琥珀色 |
| 危险/危急值 | `#dc2626` | 红色系 |
| 信息/急诊 | `#2563eb` | 蓝色系 |

### 1.2 竞品配色对照

| 系统 | 主题 | 特色 |
|------|------|------|
| **GE Centricity** | 深色为主 | 深蓝灰工具栏，黑色影像区，白色报告编辑区 |
| **Siemens syngo** | 深色主题 | 暗灰色面板，蓝色高亮，影像区极深黑 |
| **东软PACS** | 浅色为主 | 白色界面蓝色点缀，专业医疗风格 |
| **G005（当前）** | 浅色为主 | `#eff6ff`蓝色渐变头部，浅灰背景 |

**设计建议**：采用**深色工具栏 + 黑色影像区 + 白色报告编辑区**的混合方案，这是行业主流做法。

## 二、布局结构（三栏/双栏设计）

### 2.1 推荐布局：左工具栏 + 中影像区 + 右信息面板

```
┌────────────────────────────────────────────────────────────────────┐
│  顶部工具栏（检查信息 + 操作按钮）                                    │
├──────┬─────────────────────────────────────────────┬───────────────┤
│      │                                             │               │
│  左  │                                             │    右侧面板   │
│  侧  │           DICOM影像主显示区                 │   (280-320px) │
│  工  │           (深黑色背景)                       │               │
│  具  │                                             │  - 检查信息   │
│  栏  │                                             │  - 序列列表   │
│  60  │                                             │  - 报告状态   │
│  px  │                                             │  - 历史报告   │
│      │                                             │  - 模板词库   │
├──────┴─────────────────────────────────────────────┴───────────────┤
│  底部：序列缩略图条 + 报告书写区（可折叠/可全屏）                      │
└────────────────────────────────────────────────────────────────────┘
```

### 2.2 布局要素详解

| 区域 | 宽度/高度 | 样式特点 |
|------|----------|---------|
| **左侧工具栏** | 60px固定 | 深蓝灰渐变背景，图标按钮，顶部深色渐变 `primary→primaryLight` |
| **影像主显示区** | flex: 1，自适应 | `#111`极深背景，最大化影像对比度 |
| **右侧面板** | 280-320px固定 | 浅灰背景 `#f8fafc`，左侧1px灰边框阴影 |
| **底部序列栏** | 100px固定 | `#0d1117`深色背景，缩略图72×72px |
| **报告编辑区** | 可折叠，可全屏 | 白色背景，避免与影像区相互干扰 |

### 2.3 竞品布局对照

| 系统 | 布局特点 |
|------|---------|
| **GE Centricity** | 左侧工具栏60px + 中央影像 + 右侧280px信息面板 + 底部报告书写区 |
| **Siemens syngo** | 极简左侧图标栏 + 大面积影像区 + 可隐藏的右侧面板 |
| **东软** | 标准三栏，左侧检查列表，中央影像，右侧报告编辑 |
| **G005（当前）** | 左侧患者列表 + 中央影像 + 右侧信息面板，报告区在底部或模态框 |

## 三、报告书写区设计

### 3.1 报告区位置选择

**推荐方案：底部可折叠面板 + 全屏模式**

- 默认状态：报告区折叠为底部工具条（仅显示检查状态+快速按钮）
- 展开状态：报告区展开至屏幕下1/3-1/2高度
- 全屏状态：报告书写占满整个工作区（阅读报告时）

### 3.2 报告编辑区结构

```
┌─────────────────────────────────────────────────────────────────┐
│  [模板▾] [常用语] [典型征象] [历史引用] [AI辅助]    [保存] [提交] │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  检查所见：                                                     │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 多行文本编辑区，支持快捷短语插入                          │   │
│  │ - 模板自动填充                                           │   │
│  │ - 历史报告引用                                           │   │
│  │ - AI推荐语句                                             │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  诊断意见：                                                     │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                            │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  建议：                                                         │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                            │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  [报告评分] [修改痕迹] [危急值] [ICD-10]        [保存草稿] [提交] │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 编辑区核心设计要点

| 要素 | 设计建议 |
|------|---------|
| **所见/印象/建议分段** | 所见描述区高度占60%，结论区20%，建议区20% |
| **模板快捷选择** | 工具栏按钮，点击后弹窗选择模板分类（CT头部/MR脊柱等） |
| **常用语词库** | 按检查类型分类，快捷插入短语 |
| **历史报告引用** | 右侧面板显示历史报告列表，点击引用到编辑区 |
| **AI辅助推荐** | 基于检查类型+影像所见，AI推荐描述语句 |
| **文字编辑器** | 支持富文本（加粗/下划线/上下标），用于测量值标注 |
| **自动保存** | 每30秒自动保存草稿，显示保存状态 |

## 四、工具栏设计

### 4.1 左侧工具栏（影像操作）

```
┌────┐
│ 📂 │  文件（导入/导出）
├────┤
│ 🔍 │  缩放
│ ✋ │  平移
│ 🖼 │  窗口/层级（窗宽窗位）
│ 📏 │  测量（距离/角度/CT值）
│ ✏️ │  标注/画笔
│ 🖐 │  窗宽预设（骨窗/肺窗等）
├────┤
│ ⟳  │  旋转/翻转
│ 🔄 │  重置
├────┤
│ 📑 │  序列列表
│ 📝 │  报告书写
│ 📊 │  测量记录
└────┘
```

**样式规格**：
- 按钮尺寸：44×44px
- 圆角：10px
- 背景：透明 → 悬停`rgba(255,255,255,0.15)` → 激活`rgba(255,255,255,0.25)`
- 文字颜色：`rgba(255,255,255,0.7)` → `#fff`
- 按钮间距：4px
- 分割线：32px宽，1px高，`rgba(255,255,255,0.2)`

### 4.2 顶部工具栏（检查信息+操作）

```
[检查项目] [患者姓名] [设备] [检查日期]  ...  [序列: 1/20] [布局▾] [预设窗宽▾] [全屏]
```

**样式规格**：
- 背景：`#f8fafc`（浅灰）
- 下边框：`1px solid #e2e8f0`
- 内部分割：垂直灰线 `border-right: 1px solid #e2e8f0`
- 字体：标签`12px #64748b`，值`12px #1e3a5f 600`

### 4.3 窗宽窗位预设（影像工具）

| 预设名称 | 窗宽(WW) | 窗位(WL) | 按钮颜色 |

| 骨窗 | 2000 | 500 | `#f59e0b`琥珀色 |
| 肺窗 | 1200 | -600 | `#10b981`绿色 |
| 软组织 | 400 | 40 | `#8b5cf6`紫色 |
| 纵隔 | 350 | 50 | `#ec4899`粉色 |
| 脑组织 | 80 | 40 | `#3b82f6`蓝色 |
| 肝脏 | 150 | 30 | `#f97316`橙色 |
| 血管 | 300 | 100 | `#dc2626`红色 |

## 八、整体视觉风格要点

### 8.1 五大设计原则

1. **影像优先**：影像区域最大化，对比度最高，报告编辑不影响影像判读
2. **信息层级清晰**：主色用于重要操作/标题，灰色用于辅助说明
3. **效率导向**：常用功能一键触达（模板/常用语/历史引用）
4. **专业沉稳**：深蓝灰色调为主，避免过于鲜艳的配色
5. **一致性**：全系统统一的设计语言（圆角/阴影/间距/字体）

### 8.2 圆角体系

| 元素 | 圆角值 |
|------|-------|
| 按钮/输入框 | 8px |
| 卡片/面板 | 10-12px |
| 模态框 | 16px |
| 工具栏按钮 | 10px |
| 缩略图 | 6px |
| 叠加信息 | 6px |

### 8.3 阴影体系

| 元素 | 阴影值 |
|------|-------|
| 卡片默认 | `0 2px 8px rgba(30,58,95,0.08)` |

| 影像区 | `0 8px 32px rgba(0,0,0,0.5)` |

### 8.4 字体体系

| 用途 | 字号/字重 |

## 十、总结：核心设计规范

### 十大设计要点

1. **深色工具栏**：`#1e3a5f`渐变背景，白色图标，60px宽
2. **黑色影像区**：`#0a0a0a`或`#111`背景，最大化影像对比度
3. **白色报告区**：纯白编辑背景，避免与影像视觉干扰
4. **右面板280-320px**：固定宽度，浅灰背景，Tab切换
5. **五色报告状态**：草稿灰/待审琥珀/一审蓝/待签发琥珀/已签发绿
6. **七种窗宽预设**：彩色按钮（骨窗琥珀/肺窗绿/软组织紫等）
7. **四角影像叠加**：检查信息/窗宽窗位/测量数据/序列号
8. **底部序列缩略图**：100px高，72×72px缩略图
9. **模板分类体系**：按检查类型+部位二级分类，带颜色标识
10. **危急值系统**：红色高亮，即时电话确认闭环
